Lv 110

Tout le monde se spécialise : DPT mêlée/distance, Tank placement, Support Heal/Shield, etc.
Préface

Tous les stuffs sont là à titre d'exemples, et reflètent parfois mon opinion. Un Xelor et un Elio sont tous les deux des dpt distances, mais n'ont pas nécessairement les mêmes besoins; le xel aimera par exemple les PO alors que l'elio n'en a pas besoin. N'hésitez pas à adapter selon vos besoins !

Au niveau 110, il est très grandement conseillé de se spécialiser dans différents rôles (Support / Tank / DPT). Toutes les classes ont accès à leurs kits de sorts complets, les mobs commencent à infliger trop de dégats que pour que les DPT ne puisse les tanker, ce pourquoi il est important d'avoir des rôles dédiés.

De même, les combats commencent à avoir des mécaniques demandant du placement et/ou de la tankiness.

De façon simplifiée :

Pour les supports "tanks placement"

  • Pandawa (Tank + placeur)
    • Le panda peut jouer Tank + placeur; son rôle sera alors de placer les mobs de façon pratique pour les DPT, par exemple :
      • Faire des "packs" de mobs en zone pour que son Cra puisse Fleche Explo dessus
      • Prendre les mobs proche des DPT et les envoyer au loin pour les protéger
  • Sacrieur (Tank + placeur)
      • Faire des "packs" de mobs en zone pour que son Cra puisse Fleche Explo dessus (Porter/Jeter, Triple Karma Leet)
      • Tacler les mobs et les attirer/garder loin des DPT (Tonneau+ Frappe Fût)

Pour les supports "tanks soutien"

  • Osamodas (Tank + shield alliés + buff alliés + debuff mobs + placeur)
    • Shield les alliés (prendre le passif "Dévouement animal" pour mettre du shield avec les sorts terre)
    • Buff les alliés (Piqure motivante, Invocations type piou rouge)
    • Debuff les resistances des ennemis ("Cri Affaiblissant")
    • Forcer les ennemis à être tournés dos aux DPT avec le sort "Déplumage"
  • Feca (Tank + Shield alliés + Buff alliés mêlée + placeur)
    • Shield les alliés
    • Si on a un DPT mêlée => lui donner le sort Bâton pour le buff, quand il proche des mobs
    • Replacer les alliés (Frappe Tellurique)
    • Replacer les mobs (Avalanche)
  • Zobal (Tank + Shield (+heal) + buff alliés + placeur + debuff mobs)
    • Shield les alliés (Armature, Cabriole, )
    • Buff les alliés (Masque, Détraquage)
    • Replacer les alliés et les mobs (Les sorts airs : Dislocation, Poursuite, Cavalcade, ... )
    • Soigner les alliés (Signe, Rescousse, Sarabande)
    • Donner le dos des mobs aux DPT (Coup de pied fouetté)

Pour les supports "full soutien"

  • Enirispa
    • Heal les alliés (Sorts eau + les marques)
    • Buff les alliés (Revitalisation, Flamme Purificatrice, Lien d'amitié)
    • Debuff les résistances des boss (Lapino quand invoqué lorsque l'eni est en mode Contre Nature)
  • Sadida
    • Heal les alliés
    • Shield les alliés
    • Retirer des résistances aux mobs (Coup de froid , Voodool sur les boss)
    • Buff les alliés
Rappel important
  • L'enchantement de votre stuff est tout aussi important que les items. La moité de vos stats proviennent de là.
  • Tous les donjons sont faisables avec un stuff orange, les épiques et reliques donnent surtout + de PA/PM/PO.
    • La sublimation "Expert des armes légères" est de l'optimisation, et ne doit être considérée qu'après avoir obtenu votre stuff et l'avoir enchanté correctement (de préférence avec les couleurs en prévision des sublimations à poser dans le futur).
DPT distance


Lien ZenithWakfu : Build DTP Lv 110

Ce build est suffisant pour faire du S5. Il ne s'agit pas d'un build "opti", mais bien d'un build avec un bon ratio "stats/prix".

Notes :

  • Le Gelano est une très bonne relique et un excellent investissement. Vous pouvez le garder jusqu'au niveau 200 si vous souhaitez économiser vos kamas sur tous vos builds entre le lv60 et 200. Il s'agit de l'item le plus cher de ce stuff, cependant je vous conseille fortement d'en obtenir un.
  • L'Insangsé est également une très bonne épique que vous pouvez garder jusqu'au 200 si vous souhaitez économiser des kamas. (ou le changer par une Bagus Shushu au lv 125).

Améliorations possibles si kamas :

  • Remplacer le plastron par une Combinaison Lardante; il s'agit du seul plastron +1PA de la tranche
  • Remplacer le cac par un cac 1 main au choix, et jouer avec la sublimation Expert des Armes légères
  • Si vous avez besoin de PW (Xelor, Cra) vous pouvez équiper une Gelature en ceinture si vous en ressentez le besoin
