Jouer sous Linux

Preface

Il est tout à fait possible de jouer à Dofus ou Wakfu sous Linux, et ce assez facilement. En effet, Ankama met à disposition des fichers "AppImage", lesquels vous pouvez considérer un peu comme des executables sur Windows (en vérité c'est pas du tout la même chose, mais dans le cadre de ce tuto peu importe).

Ce tutoriel est rédigé sous Ubuntu LTS (Actuellement v24), mais devrait fonctionner pour toutes les distros supportant le format AppImage.

Installation

Rendez-vous sur l'un des sites officiels d'Ankama :

  • https://www.dofus.com
  • https://www.wakfu.com

Et rendez-vous sur la page de download (ou cliquez "Jouer" en haut à droite)

  • https://www.dofus.com/fr/mmorpg/telecharger
  • https://www.wakfu.com/fr/mmorpg/telecharger
NOTE - Peu importe lequel vous téléchargez, les deux font la même chose : ils permettent d'installer l'Ankama Launcher.

Selon lequel vous avez téléchargé, vous avez maintenant un ficher finissant par ".AppImage" .

  • "Dofus 3.0-Setup-x86_64.AppImage" si vous avez suivi le lien de Dofus
  • "Wakfu-Setup-x86_64.AppImage" si vous avez suivi le lien de Wakfu

Malgré leurs différences de nom, il s'agit en réalité des mêmes fichiers, qui font la même chose : installer l'Ankama launcher.

Vous avez maintenant votre fichier. Mais comme Linux a tendance à être + sécurisé que Windows, il faut faire une denière chose :

  • Click droit sur le fichier -> Propriétés -> Autoriser l'exécution du fichier comme un programme
Fichier .AppImage
Exemple avec le fichier Wakfu.
Permissions
Il faut cocher "Exécuter en tant que programme"/"Executable as Program"

Voilà, vous pouvez maintenant simplement double-click le fichier pour lancer l'Ankama launcher ! Là, c'est comme d'habitude : sélectionnez le jeu de votre choix, et cliquez sur installer.

Tips et tricks additionels

Lancer en double-cliquant fonctionne, mais on peut gagner un peu de confort au quotidien.

  • en permettant d'accèder via le menu des applications
    • ce qui permet aussi de chercher rapidement (touche super puis taper le nom de l'app)
  • en ajoutant le launcher dans le dock (le pin au dock)

Ca tombe bien, les deux sont liés, et je vous propose ma solution !

D'abord par facilité je vais renomer le fichier en AnkamaLauncher.AppImage . Comme ça c'est plus simple et plus parlant.

Ensuite pour m'y retrouver facilement, je vais bouger le fichier AppImage dans un dossier dédié (laisser trainer des fichiers dans downloads, c'est pas ouf). Peu importe où vous le mettez, tant que vous savez où vous l'avez mis.

Maintenant, notez le path entier (absolu) de votre AppImage quelque part, on va en avoir besoin. (c'est celui qui est indiqué dans la barre du haut, suivi par le nom de votre fichier).

    Par exemple si mon utilisateur s'appelle Aekami et que j'ai mis mon AppImage dans un dossier "Ankama_things":
  • '/home/aekami/Workspace/Ankama_things/AnkamaLauncher.AppImage'

Allez dans votre fichier '/home/[VotreNomDUtilisateur]/.local/share/applications' et créez un nouveau fichier que vous nommez "AnkamaLauncher.desktop" .

Ouvrez le fichier, et ajoutez ceci, en modifiant selon l'emplacement des différents fichiers chez vous :

[Desktop Entry]
Name=Ankama Launcher Custom
Exec=/home/aekami/Workspace/Ankama_things/AnkamaLauncher.AppImage --no-sandbox
Icon=/home/aekami/Workspace/Ankama_things/zaap.png
Type=Application
Comment=Ankama portal and game launcher
Categories=Game;

Pour la ligne exec=, remplacez le path en gras par celui de votre AppImage. (Celui qu'on a noté à l'étape juste avant).

Pour la ligne Icon=, vous pouvez soit la retirer, soit mettre le path d'une icone, qui sera affiché pour votre app.
