Sites à connaître

ZenithWakfu - https://www.zenithwakfu.com

Builder de stuff, même idée que Dofusbook, DofusRoom, DofusPlanner, etc. mais pour Wakfu.

Il s'agit du seul builder de stuff Wakfu. En général quand vous chercherez des stuffs, ce sont des liens vers ce site que l'on vous proposera. C'est également des liens vers ce site que vous trouverez dans mes pages de stuffs.

MethodWakfu - https://methodwakfu.com

Guides pour les quêtes du jeu. Un peu le DofusPourLesNoobs de Wakfu

Il s'agit du principal guide des quêtes pour Wakfu. Il est relativement complet, même si parfois un brin outdated.

Les quêtes principales sont gardées à jour, et une partie des donjons ont leurs fonctionnements, mécaniques, et mobs d'expliqués. Si vous êtes bloqués dans une quête ou vous posez des questions sur une mécanique du jeu, il est probable qu'une explication soit quelque part sur le site.

Encyclopédie officielle - https://www.wakfu.com/fr/mmorpg/encyclopedie
Qui contient également le bestiaire - https://www.wakfu.com/fr/mmorpg/encyclopedie/monstres

Il s'agit de l'encyclopédie officielle (par Ankama, sur le site officiel), qui regroupe entre autres :

  • tous les monstres du jeu
    • avec les drops possibles et le taux de drop
  • tous les items
    • avec leur craft si connu
    • avec les mobs sur lesquels l'item peut être drop, si l'item est droppable
Organiser vos crafts métier - Craftkfu

Une app en ligne qui vous permet de selectionner un item que vous souhaitez craft, et qui vous fournit la liste des ressources pour le craft. Particulièrement pratique lorsque vous avez plusieurs items à craft.

Site de Nozadah - https://wakfu.guide

Un peu de tout sur Wakfu, de bons guides et tutos.

Regardez plutôt les vidéos tutos, qui ont tendance a être plus claires que les articles.

Néanmoins certaines pages restent très intéressantes, telles que les pages sur les sublimations et les pages sur les conseils de stuffs.
