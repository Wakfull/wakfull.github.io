Mécaniques spécifiques de la classe

L'Eni est une classe centrée sur le soin, et le soutien aux aliés, mais également capable de mettre des dégats quand il le souhaite.

Les mécaniques de classes principales sont les suivantes :

  • La gestion du passif "Propagateur"
  • L'état contre-nature
Et les mécaniques secondaires :
  • Le lapino
  • Les sorts sont améliorés lorsque l'eni a + de 80% de ses points de vies.
  • Passivement, l'eni augmente les pv totaux de ses aliés

Récupération de PW

La façon dont les enis récupèrent des PW est :
